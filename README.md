A toy compiler which targets to LLVM

## Requirements

- Dotnet core 3.1
- LLVM 9+
- Rust edition 2018

## Libraries

- Pidgin: Parser Combinator
- Fody: codegen for dotnet
- LLVMSharp: LLVM binding of C#
- Cake: Makefile for C#
