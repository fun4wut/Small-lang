# [WIP] Kumiko-lang Intro

![](https://gofun4-pic.oss-cn-hangzhou.aliyuncs.com/422a92f0434f1a69.gif)

一个玩具语言，编译到LLVM

## Requirements

- Dotnet core 3.1
- LLVM 9+

## 所使用的库

- Pidgin：Parser Combinator
- Fody：dotnet代码的codegen
- LLVMSharp：LLVM的C# binding
